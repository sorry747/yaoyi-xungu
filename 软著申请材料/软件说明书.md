## 一、软件概述

### 1.1 软件名称

- **软件全称**：瑶医古语AI智能训诂系统
- **软件简称**：瑶医训诂系统
- **英文名称**：Yao Medicine Ancient Text AI Exegesis System
- **版本号**：V1.0

### 1.2 软件简介

本系统是一款基于人工智能技术的瑶医古语训诂工具，采用LangChain框架开发，集成豆包大模型，能够对瑶医养生古语进行六维度专业训诂分析。系统结合传统训诂学理论与现代AI技术，为古汉语教学、瑶医文化研究、非遗文献整理提供智能化解决方案。

### 1.3 设计目标

1. **学术价值**：为训诂学研究提供AI辅助工具
2. **文化传承**：促进瑶医养生文化的数字化传承
3. **教学辅助**：为高校古汉语教学提供智能化工具
4. **技术创新**：探索AI在传统训诂学中的应用

### 1.4 应用领域

- 高等教育（古汉语教学）
- 文化研究（瑶医文化、非遗保护）
- 文献整理（古籍数字化）
- 养生传播（传统文化普及）

### 3.2 智能训诂模块

#### 3.2.1 模块概述

这是系统的核心模块，负责对用户输入的瑶医古语进行专业的训诂分析，输出六维度详细结果。

#### 3.2.2 功能流程

```
用户输入古语
      ↓
  文本预处理
      ↓
  LLM调用分析
      ↓
  结果格式化
      ↓
  结果保存到语料库
      ↓
  结果展示
```

#### 3.2.3 六维度训诂

**（1）形训子模块**

- **功能**：分析字形结构
- **实现方式**：
  - 识别生僻字、异体字、通假字
  - 分析字形的文化内涵
  - 提供字形变体信息

**（2）音训子模块**

- **功能**：分析字音
- **实现方式**：
  - 提供标准普通话读音（带拼音）
  - 标注古音、方言音（如瑶语发音）
  - 分析音韵关系

**（3）义训子模块**

- **功能**：解释词义
- **实现方式**：
  - 逐词训诂，解释每个词的含义
  - 分析句子的核心语义
  - 揭示词义的引申、比喻用法

**（4）文化溯源子模块**

- **功能**：追溯文化背景
- **实现方式**：
  - 分析古语的文化来源
  - 关联瑶医理论体系
  - 说明传承路径

**（5）草药关联子模块**

- **功能**：识别关联草药
- **实现方式**：
  - 提取古语中的草药名称
  - 说明每味草药的功效、性味、归经
  - 提供配伍禁忌和注意事项

**（6）今译注释子模块**

- **功能**：现代汉语翻译
- **实现方式**：
  - 准确翻译为现代汉语
  - 保留原文的文化内涵
  - 必要时添加注释说明

## 四、核心算法

### 4.1 训诂分析算法

#### 4.1.1 算法概述

基于大语言模型的训诂分析算法，通过Prompt Engineering引导模型输出结构化的训诂结果。

#### 4.1.2 算法流程

```
1. 用户输入古语文本
   ↓
2. 构建System Prompt
   （定义训诂规则和输出格式）
   ↓
3. 调用LLM API
   ↓
4. 解析模型输出
   ↓
5. 格式化为JSON
   ↓
6. 返回给用户
```

#### 4.1.3 Prompt设计

```python
SYSTEM_PROMPT = """
你是一位精通瑶医文化与训诂学的专家。
```

### 4.2 语料库检索算法

#### 4.2.1 算法概述

基于关键词匹配的语料库检索算法，支持模糊查询。

#### 4.2.2 算法实现

```python
def search_corpus(corpus: dict, keyword: str) -> list:
    """语料库检索"""
    results = []
    for item in corpus["corpus"]:
        # 匹配原文
        if keyword in item["original_text"]:
            results.append(item)
        # 匹配今译注释
        elif keyword in item["今译注释"]:
            results.append(item)
    return results
```

#### 4.2.3 性能优化

- 使用列表推导式提高效率
- 支持大小写不敏感搜索
- 限制返回结果数量（最多50条）

### 4.3 统计分析算法

#### 4.3.1 算法概述

对语料库数据进行多维统计分析，提取关键指标。

#### 4.3.2 算法实现

```python
def analyze_statistics(corpus: dict) -> dict:
    """统计分析"""
    total = len(corpus["corpus"])
    herbs = set()
    cultures = set()

    for item in corpus["corpus"]:
        # 提取草药
        herbs.update(extract_herbs(item["草药关联"]))
        # 提取文化来源
        cultures.update(extract_cultures(item["文化溯源"]))

    return {
        "古语总数": total,
        "涉及草药": sorted(list(herbs)),
        "文化来源": sorted(list(cultures))
    }
```
