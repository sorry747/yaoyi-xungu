# 瑶医古语AI智能训诂系统 V1.0 用户手册

---

## 一、软件简介

### 1.1 软件名称
瑶医古语AI智能训诂系统 V1.0

### 1.2 软件功能
本系统是一款基于人工智能技术的瑶医古语训诂工具，能够对瑶医养生古语进行六维度专业训诂分析，包括形训、音训、义训、文化溯源、草药关联、今译注释。

### 1.3 适用对象
- 高校古汉语教学师生
- 瑶医文化研究者
- 非遗文献整理工作者
- 养生文化爱好者

### 1.4 运行环境
- 操作系统：Windows 10/11、macOS、Linux
- 浏览器：Chrome、Edge、Firefox、Safari
- 网络环境：需连接互联网（调用LLM API）

### 3.2 智能训诂功能

#### 3.2.1 功能概述
对用户输入的瑶医古语文本进行专业训诂分析，输出六维度详细结果。

#### 3.2.2 使用步骤

**步骤1**：进入智能训诂界面
- 点击左侧导航栏的"🔍 智能训诂"

**步骤2**：输入瑶医古语
- 在输入框中输入需要训诂的古语文本
- 示例："香囊佩胸前，祛湿又散寒"

**步骤3**：开始训诂
- 点击"开始训诂"按钮
- 系统自动调用AI模型进行分析
- 等待10-30秒

**步骤4**：查看结果
- 系统显示完整的六维度训诂结果
- 包括：形训、音训、义训、文化溯源、草药关联、今译注释

**步骤5**：下载结果
- 点击"下载JSON格式"按钮
- 保存训诂结果到本地

#### 3.2.3 训诂结果说明

**形训**：
- 分析字形结构
- 识别生僻字、异体字、通假字
- 说明字形的文化内涵

**音训**：
- 提供标准普通话读音（带拼音）
- 标注古音、方言音（如瑶语发音）
- 分析音韵关系

**义训**：
- 逐词训诂，解释每个词的含义
- 分析句子的核心语义
- 揭示词义的引申、比喻用法

**文化溯源**：
- 追溯文化背景
- 说明传承来源
- 关联瑶医理论

**草药关联**：
- 识别关联的瑶医草药
- 说明每味草药的功效、性味、归经
- 提供配伍禁忌和注意事项

**今译注释**：
- 准确的现代汉语翻译
- 保留原文的文化内涵
- 必要时添加注释说明

#### 3.2.4 示例

**输入**：
```
香囊佩胸前，祛湿又散寒
```

**输出**：
```json
{
  "原文": "香囊佩胸前，祛湿又散寒",
  "形训": "字形分析内容...",
  "音训": "字音分析内容...",
  "义训": "语义解释内容...",
  "文化溯源": "文化背景分析...",
  "草药关联": "草药关联分析...",
  "今译注释": "现代汉语翻译..."
}
```

## 四、注意事项

### 4.1 使用注意事项

1. **网络要求**：
   - 系统需要稳定的网络连接
   - 训诂分析需要调用远程LLM API
   - 建议使用稳定的WiFi或有线网络

2. **输入规范**：
   - 输入的古语应为完整的句子或短语
   - 避免输入过长文本（建议不超过100字）
   - 支持中文输入

3. **结果保存**：
   - 训诂结果会自动保存到语料库
   - 建议定期下载重要结果备份
   - 语料库存储在本地

### 4.2 性能提示

1. **响应时间**：
   - 简单古语：10-20秒
   - 复杂古语：20-40秒
   - 受网络状况影响

2. **并发限制**：
   - 建议同时不超过3个训诂请求
   - 避免频繁重复提交相同内容

### 4.3 数据安全

1. **隐私保护**：
   - 用户输入的古语会保存在本地语料库
   - 不会上传到第三方服务器（除LLM API调用外）
   - 建议定期清理语料库

2. **数据备份**：
   - 建议定期备份语料库文件
   - 语料库文件位置：`assets/yaoyi_corpus.json`

## 五、常见问题

### Q1：训诂分析失败怎么办？

**A**：
- 检查网络连接
- 检查输入的古语格式
- 重新点击"开始训诂"
- 如仍失败，联系技术支持

### Q2：为什么有些古语训诂不准确？

**A**：
- AI模型的准确率约为85%
- 建议人工复核重要内容
- 可以通过"文化溯源"等维度辅助判断
- 系统会不断优化提升准确率

### Q3：如何清除语料库？

**A**：
- 语料库文件位置：`assets/yaoyi_corpus.json`
- 删除或重命名该文件即可
- 重新启动系统后，语料库将重置

### Q4：系统支持哪些浏览器？

**A**：
- Chrome（推荐）
- Edge
- Firefox
- Safari
- 其他现代浏览器

### Q5：如何在手机上使用？

**A**：
- 在电脑上启动系统
- 查看本机IP地址
- 在手机浏览器输入：`http://本机IP:8501`
- 例如：`http://192.168.1.100:8501`
