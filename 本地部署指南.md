## 🎯 适用场景

- 无法访问外网
- 本地演示

## 📋 部署准备

### 1. 环境要求

- Python 3.8 或更高版本
- 4GB 或以上内存
- 10GB 或以上磁盘空间

### 2. 依赖安装

```bash
# 安装Python依赖
pip install -r requirements.txt
```

---

## 🚀 部署步骤

### 方法1：本地运行（最简单）

#### 步骤1：启动服务

```bash
cd /workspace/projects
streamlit run app.py
```

#### 步骤2：访问地址

在浏览器打开：
```
http://localhost:8501
```

---

### 方法2：局域网访问（适合答辩展示）

#### 步骤1：启动服务

```bash
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

#### 步骤2：查看本机IP

```bash
# Windows
ipconfig

# Linux/Mac
ifconfig
```

找到你的IP地址，例如：`192.168.1.100`

#### 步骤3：访问地址

在局域网内其他设备浏览器打开：
```
http://192.168.1.100:8501
```

**优点**：
- 手机、平板、电脑都能访问
- 无需外网
- 适合答辩现场展示

---

### 方法3：打包成独立可执行文件

#### 步骤1：安装PyInstaller

```bash
pip install pyinstaller
```

#### 步骤2：打包

```bash
pyinstaller --onefile --windowed app.py
```

#### 步骤3：运行

打包后，在`dist/`目录下找到：
- Windows：`app.exe`
- Linux：`app`
- Mac：`app.app`

双击即可运行，无需安装Python！

## 📱 软著申请专用说明

### 软件基本信息

- **软件全称**：瑶医古语AI智能训诂系统
- **软件简称**：瑶医训诂系统
- **版本号**：V1.0
- **软件分类**：应用软件
- **行业分类**：教育软件

### 功能描述

本系统是一款基于人工智能技术的瑶医古语训诂工具，主要功能包括：

1. **智能训诂分析**：对瑶医养生古语进行六维度专业训诂
2. **语料库管理**：自动保存和查询训诂结果
3. **统计分析**：提供语料库数据统计功能
4. **可视化展示**：Web界面，操作简单直观

### 技术特点

- 基于LangChain框架开发
- 集成豆包大模型
- 采用Streamlit框架构建Web界面
- 支持多维度训诂分析

### 应用场景

- 高校古汉语教学辅助
- 瑶医文化研究
- 非遗文献整理
- 养生文化传播

## 🔧 故障排除

### 问题1：启动失败

**解决方法**：
```bash
# 检查Python版本
python --version

# 重新安装依赖
pip install -r requirements.txt --upgrade
```

### 问题2：无法访问

**解决方法**：
```bash
# 检查端口是否被占用
netstat -an | grep 8501

# 更换端口
streamlit run app.py --server.port 8502
```

### 问题3：功能异常

**解决方法**：
- 检查网络连接
- 检查API密钥配置
- 查看日志文件
